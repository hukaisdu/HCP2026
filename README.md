# HCP 2026

第九届难解问题的理论、算法与应用研讨会官网初稿，采用山东大学风格的静态单页站点。

## 文件

- `index.html`：页面结构与会议内容
- `en.html`：英文版页面
- `styles.css`：山东大学风格视觉样式
- `script.js`：移动端导航与当前栏目高亮
- `assets/`：山东大学官网素材与页面图片

## 本地预览

直接在浏览器打开 `index.html` 即可预览。
